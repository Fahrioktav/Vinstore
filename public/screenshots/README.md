# PWA Screenshots

Place app screenshots here for better PWA install experience.

## Required Sizes
- **Mobile**: 540x720px (portrait) - home.png
- **Desktop**: 1280x720px (landscape) - Optional

## How to Create Screenshots

1. Open Vinstore in browser
2. Set browser window to mobile size (DevTools → Toggle Device Toolbar)
3. Take screenshot of:
   - Homepage
   - Products page
   - Store page
   - Checkout flow (optional)

4. Resize to 540x720px
5. Save as `home.png` in this folder

## Tools
- Chrome DevTools: Capture device screenshot
- Online: https://www.screely.com/
- Browser Extension: GoFullPage, Awesome Screenshot

Screenshots improve the install banner and make the app more appealing!
